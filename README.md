# Music Recommender

A machine learning project that recommends music genres based on a person's age and gender, built using Python and scikit-learn.

## What it does

- Takes a user's age and gender as input
- Predicts the most suitable music genre using a Decision Tree classifier
- Achieves 75%+ model accuracy on the dataset
- Visualises the decision tree as a `.dot` file

## Tech Stack

- Python
- Pandas
- scikit-learn
- Jupyter Notebook

## How to run

1. Clone the repo
   git clone https://github.com/gagankrishnaa/music-recommender.git

2. Install dependencies
   pip install pandas scikit-learn jupyter

3. Open the notebook
   jupyter notebook music_recommender.ipynb
